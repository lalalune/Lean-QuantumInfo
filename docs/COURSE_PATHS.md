# Course Paths

The course import ladder is:

1. `QuantumInfo.CoursePrelude.Level1`
2. `QuantumInfo.CoursePrelude.Level2`
3. `QuantumInfo.CoursePrelude.Level3`

## Level 1

Focus:
- foundational mathematics
- units and dimensions
- classical probability/distributions

Imports:
- `Mathematics`
- `Units`
- `ClassicalInfo`

## Level 2

Focus:
- finite-dimensional QM/QI
- channels, entropies, coding statements
- QEC foundations

Imports:
- `QuantumMechanics`
- `QuantumInfo`
- `QEC`

## Level 3

Focus:
- relativity and QFT
- particles/stat mech/cosmology/condensed matter interfaces

Imports:
- `Relativity`
- `QFT`
- `Particles`
- `StatMech`
- `Cosmology`
- `CondensedMatter`
